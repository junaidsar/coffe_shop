# README

# Requirements

* Ruby version: `2.7.6`
* Database: `Sqlite3`
* Rails: `7.0.3`

# How to run the project

* bundle install
* rake db:create
* rake db:migrate
* rake db:seed
* rails s


# How to run the test suite
	I used minitest for the item and order tests, You can use following command to execute the tests
	`rails test`
